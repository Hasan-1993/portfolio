Bookstore Management App

This is a Full Stack project of a bookstore management app. The frontend, made using Vite, React and TailwindCSS, shows the book inventory, and options to perform CRUD. While the Backend, created using Node, Express and MongoDB interacts with the Database to make changes in the inventory.

Cuddle Clones Website

It is a Frontend-only project of a business that manufactures stuffed animal replicas of real life pet's through their images. The project uses vanilla stack of HTML, CSS and JavaScript. However, the detail given to each component is extensive, and the website is fully-responsive for any screen size.

×

News App Website

This web app shows the latest news fetched form a third-party API. It maps all the news components dynamically using Bootstrap features, creating more as you go along. It has several React & Bootstrap features like infintie scroll, top loading bar, state management and hooks.

Landing Page - Tinder for Pet Dogs

This is a website for matching pet dogs to their potential Tinder Mates. It has a matching app on Andriod and Apple, with Free, Paid and Premium features. The project does not use any framewrks, and is fully responsive using only the Vanilla languages. Overall, its a small and simple website.
